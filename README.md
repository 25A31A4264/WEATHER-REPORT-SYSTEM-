🌦️ Weather Report System (C++)

📌 Overview

The Weather Report System is a simple C++ console-based application that allows users to store and view weather data for different cities.
It simulates basic functionality of real-world weather apps like Google Weather.

---

✨ Features

- Add weather details (City, Temperature, Condition)
- View all stored weather data
- Data stored using file handling
- Simple and user-friendly CLI interface

---

🛠️ Technologies Used

- C++
- File Handling ("fstream")
- Object-Oriented Programming (OOP)

---

📂 Project Structure

weather-system/
│── main.cpp
│── weather.txt
│── README.md

---

▶️ How to Run

Step 1: Clone the repository

git clone https://github.com/your-username/weather-system.git
cd weather-system

Step 2: Compile the program

g++ main.cpp -o weather

Step 3: Run the program

./weather
